# Etsy Market Research Collector

Etsy Market Research Collector is a private research tool for organizing Etsy market data through Etsy Open API v3.

## Purpose

The application helps with product research and market analysis by collecting and organizing publicly available Etsy data such as:

- Active listing information
- Shop profile information
- Listing images and videos when available through the official API
- Listing and shop reviews when available through the official API
- Etsy taxonomy/category data

## API Usage

The application uses Etsy Open API v3 official endpoints. It is designed to respect Etsy's documented API rate limits and request standards.

The application does not:

- Scrape Etsy web pages
- Bypass login, CAPTCHA, bot protection, or access controls
- Use proxy rotation to evade rate limits
- Ask users to share API keys publicly
- Collect private shop data without OAuth authorization

## Users

This tool is currently intended for private use by the developer and colleagues for product research and internal analysis.

## Data Handling

Collected data is used for internal product research and analysis. Private shop data, if used in the future, will require OAuth authorization from the shop owner and will be handled according to Etsy API Terms and applicable privacy requirements.

## Contact

For questions about this application, contact the developer through the Etsy developer account associated with the API application.
