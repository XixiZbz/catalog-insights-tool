# Catalog Insights Tool

Catalog Insights Tool is a private internal tool for organizing catalog data through Open API v3.

## Purpose

The application helps with product and catalog analysis by collecting and organizing publicly available catalog data such as:

- Active listing information
- Shop profile information
- Listing images and videos when available through the official API
- Listing and shop reviews when available through the official API
- Etsy taxonomy/category data

## API Usage

The application uses Open API v3 official endpoints. It is designed to respect documented API rate limits and request standards.

The application does not:

- Scrape web pages
- Bypass login, CAPTCHA, bot protection, or access controls
- Use proxy rotation to evade rate limits
- Ask users to share API keys publicly
- Collect private shop data without OAuth authorization

## Users

This tool is currently intended for private use by the developer and colleagues for product research and internal analysis.

## Data Handling

Collected data is used for internal product research and analysis. Private shop data, if used in the future, will require OAuth authorization from the shop owner and will be handled according to API Terms and applicable privacy requirements.
